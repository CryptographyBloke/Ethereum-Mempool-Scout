# 题目
分治策略——整数因子分解问题
## 一、问题背景
在数论中，一个正整数的因子分解（Factorization）是指将其写成若干个正整数的乘积。与唯一的质因数分解不同，整数因子分解不要求因子必须为质数，且因子的排列顺序不同会被视为不同的分解式。这是一个典型的递归结构问题：为了求得 $n$ 的分解式，我们可以先确定第一个因子 $i$，然后递归地求解 $n/i$ 的所有分解方案。
## 二、问题描述
大于 1 的正整数 **n** 可以分解为：**n = x1 × x2 × ... × xm**。

对于给定的正整数 **n**，计算 **n** 共有多少种不同的分解式。

例如：当 **n = 12** 时，有 8 种不同的分解式：
1. **12 = 12**
2. **12 = 6 × 2**
3. **12 = 4 × 3**
4. **12 = 3 × 4**
5. **12 = 3 × 2 × 2**
6. **12 = 2 × 6**
7. **12 = 2 × 3 × 2**
8. **12 = 2 × 2 × 3**
## 三、设计思路
本问题采用分治与递归的思想：分解模型：对于 $n$，第一个因子 $i$ 的可选范围是 $[2, n]$。分治逻辑：如果我们选择了 $i$ 作为第一个因子，且 $n$ 能被 $i$ 整除，那么剩下的部分 $n/i$ 构成了一个规模更小的子问题。将 $i$ 与 $n/i$ 的每一种分解结果进行组合，即可得到 $n$ 的部分分解式。递归终点：当 $n=1$ 时，意味着当前路径已经完成了一次完整的分解，返回一个包含空列表的集合（作为递归组合的基础）。注意：为了输出所有组合，我们需要在递归过程中维护一个路径列表，记录每一步选择的因子。

## 四、 输入输出描述（测试要求）
### 输入格式及描述
输入格式及描述输入一个大于 1 的正整数 $n$。输出描述及描述逐行输出所有不同的分解式。格式要求：分解X: n=因子1x因子2x...（注意：若分解式只有一个因子即 $n$ 本身，直接输出 $n=n$）。最后一行输出总数。
### 输出描述及描述

### 输入样例1
```in 
12
```

### 输出样例1
```out 
分解1: 12=12
分解2: 12=6x2
分解3: 12=4x3
分解4: 12=3x4
分解5: 12=3x2x2
分解6: 12=2x6
分解7: 12=2x3x2
分解8: 12=2x2x3
12共有8种不同的分解式
```
设计说明：

解析：本样例展示了全排列视角的整数分解。12 可以被分解为单因子 (12)；双因子 (6×2, 4×3)；三因子 (3×2×2)。由于题目规定“因子的排列顺序不同会被视为不同的分解式”，所以 6×2 和 2×6 分别计入，3×2×2 也产生了三种不同位置的排列组合。总计 1+2+2+3 = 8 种不同的分解式。

### 输入样例2

```in 
6
```
### 输出样例2
```out 
分解1: 6=6
分解2: 6=3x2
分解3: 6=2x3
6共有3种不同的分解式
```
设计说明：

解析：6 的真因子只有 2 和 3。它可以不进一步分解（即自身 6=6）；或者分解为 2×3。同样考虑到排列顺序的影响，2×3 和 3×2 是两种不同的结果。因此总共有 1(自身) + 2(双因子排列) = 3 种分解式。

### 输入样例3

```in 
11
```
### 输出样例3
```out 
分解1: 11=11
11共有1种不同的分解式
```

设计说明：解析：本样例测试的是质数的边界情况。由于 11 是质数，除了 1 和它本身之外没有其他正整数因子，而题目要求因子的取值范围是 $[2, n]$（不包含 1），因此 11 无法进一步拆分。唯一的分解式就是它自身。
### 输入样例4

```in 
8
```
### 输出样例4
```out 
分解1: 8=8
分解2: 8=4x2
分解3: 8=2x4
分解4: 8=2x2x2
8共有4种不同的分解式
```
设计说明：解析：8 作为一个纯 2 的幂次方数 ($2^3$)。可以分解为自身 (8)；双因子组合 (4×2) 及其排列 (2×4)；以及最底层的质因子组合 (2×2×2)。其中 2×2×2 由于三个因子完全相同，不存在其它排列方式，故计为 1 种。总共为 1+2+1 = 4 种。

### 输入样例5

```in 
16
```
### 输出样例5
```out 
分解1: 16=16
分解2: 16=8x2
分解3: 16=4x4
分解4: 16=4x2x2
分解5: 16=2x8
分解6: 16=2x4x2
分解7: 16=2x2x4
分解8: 16=2x2x2x2
16共有8种不同的分解式
```

设计说明：解析：16 是 $2^4$，它的分解层级更深。分解形式包括：单因子：16 （1种）双因子：8×2, 2×8, 4×4 （3种，其中 4×4 因子相同只有1种排列）三因子：4×2×2 及其排列 2×4×2, 2×2×4 （3种）四因子：2×2×2×2 （1种，因子全部相同）累加后一共正好是 1 + 3 + 3 + 1 = 8 种不同的分解式。
